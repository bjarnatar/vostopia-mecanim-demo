Vostopia 3rd Personi Controller Example
=======================================

This example shows how to use the Character Controller package to get the vostopia avatars
walking around in a very simple scene.

The AvatarController is outfitted with a ThirdPersonController, supplied by the standard asset
package "Character Controller". It is set up with animations and walk parameters to move the 
avatar around and play animations on it.

Requirements
-----------
This example relies on the standard asset package "Character Controller" being 
imported in your project. If you get lots of warnings about missing components,
you most likely does not have the Character Controller classes.

You can import the Character Controller packge from the Assets -> Import Package menu.
